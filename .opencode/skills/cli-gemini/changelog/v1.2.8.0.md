## Summary

The cli-gemini prompt quality card had grown to 89 LOC by accumulating guidance that is already maintained authoritatively in sk-prompt. The card is now a delegating mirror: it retains Gemini-specific dispatch notes and points readers to the canonical source for framework choice, scoring, and composition rules. SKILL.md picks up the 3-tier prompt-composition precedence rule (framework > anti-hallucination wording > density) and a cross-link to sk-prompt-small-model so small-model dispatches from cli-gemini follow the same hub-level guidance.
