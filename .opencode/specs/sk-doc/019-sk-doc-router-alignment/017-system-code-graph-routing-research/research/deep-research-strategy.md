- Reading the exact scenario contracts/prompts, then applying the benchmark authoring and parser contracts, separated resource existence from a real routing decision. (iteration 4)

- Mechanical set comparison exposed the 53/35/18/0 partition; tracing generator -> replay -> scorer exposed failures path-existence checks miss. (iteration 6)
