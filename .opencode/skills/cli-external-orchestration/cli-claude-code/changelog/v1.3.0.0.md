This release condenses `SKILL.md` by relocating reference-table duplication into pointers to the ALWAYS-loaded `references/cli-reference.md`. No routing behavior, rule, model, or contract changed. Every dispatch rule, the model roster, the smart-router dictionaries, the self-invocation guard, and the OAuth auth pre-flight are preserved verbatim.

---

## What Changed

#### SKILL.md Condensation

- **Reference tables relocated to pointers** -- the flag glossary, the unique-capabilities table, the essential-command examples, and the troubleshooting table were removed from `SKILL.md` because they duplicate `references/cli-reference.md`, which is ALWAYS loaded on every invocation. `SKILL.md` now carries a compact Model Selection roster, a compact Agent Delegation table, and a Dispatch-Critical Gotchas subsection (print-mode default, `--permission-mode plan` vs `bypassPermissions`, no `--search` flag, `$CLAUDECODE` nesting check) plus pointers.
- **Model and agent tables compacted** -- the Model Selection table drops the redundant use-case prose (pointing to `cli_reference.md` §6, which lists the current-generation IDs), and the Agent Delegation table drops the per-row invocation-pattern column (pointing to `agent_delegation.md`).

#### Preserved Verbatim

- All 13 ALWAYS rules, 4 NEVER rules, 4 ESCALATE conditions, and the Memory Handback protocol.
- The `INTENT_SIGNALS` / `RESOURCE_MAP` / `LOADING_LEVELS` router dictionaries, the self-invocation guard, and the OAuth auth pre-flight (`claude auth login` / `setup-token`).
- The default-invocation override table.
