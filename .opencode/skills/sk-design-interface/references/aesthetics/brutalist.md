# Brutalist Preset

A compact visual-direction cue for raw, mechanical interfaces that feel like industrial print, telemetry, or a declassified operating console.

---

## 1. OVERVIEW

### Purpose

Use for raw, mechanical interfaces that should feel like industrial print, telemetry, or a declassified operating console.

### Usage

Apply this preset as a citeable starting cue, then adapt it to the product context instead of treating it as a fixed template. Use one preset as the dominant aesthetic per direction; if combining cues, name this as the primary preset and keep borrowed cues sparse.

---

## 2. PALETTE

- Light mode: matte paper `#f4f4f0` or `#eae8e3`, carbon ink `#050505` to `#111111`, one hazard red `#e61919` or `#ff2a2a`.
- Dark mode: CRT black `#0a0a0a` or `#121212`, white phosphor `#eaeaea`, the same red accent.
- Optional terminal green `#4af626` for one status readout only.

---

## 3. TYPE

- Macro type is oversized, uppercase, heavy neo-grotesque: Archivo Black, Monument Extended, Inter Black, or similar.
- Use tight tracking around `-0.03em` to `-0.06em` and compressed line height around `0.85` to `0.95`.
- Metadata, navigation, and coordinates use small monospace from `10px` to `14px` with wide tracking.

---

## 4. SHAPE AND LAYOUT

- Build with strict CSS grid, visible 1px or 2px compartments, and hard 90 degree corners.
- Alternate dense technical clusters with large negative-space fields around macro typography.
- Use ASCII brackets, crosshairs, barcodes, registration marks, or revision labels as structural devices.

---

## 5. MOTION AND TEXTURE

- Motion is abrupt and operational: short state snaps, terminal cursor pulses, data refresh ticks.
- Add physical grain through low-opacity noise, halftone, scanline, or one-bit dither effects.
- Avoid gradients, soft shadows, rounded corners, glass, or generic SaaS polish.
