# v1.0.0.2 - designer-skills-main build/visual adoption

**Released:** 2026-06-27

This release adopts the small, build-facing slice of a second external corpus — the `designer-skills-main` suite (nine plugins, roughly ninety-six skills spanning the whole design lifecycle) — into the design family, and deliberately leaves the rest out. The corpus is mostly lifecycle work (research, strategy, design-ops, governance) that a taste-led build/visual skill should not own; a 17-iteration study (including a cross-model completeness sweep) found the adoptable core is small and lands entirely in the existing five modes. No new mode was added, no second scoring model was introduced, and the hub stays routing-only. All edits are additive guidance; nothing changes runtime behavior.

> Source research: `.opencode/specs/skilled-agent-orchestration/154-sk-design-parent/024-designer-skills-research/research/research.md`
> Build phases: `025-audit-adoption`, `026-interface-motion-adoption`, `027-foundations-adoption` (Level 3 each)

---

## Audit

The highest-leverage adoption strengthened the audit mode without changing how it scores or routes.

#### Visual-critique crosswalk
The visual-critique plugin's seven screen-critique dimensions (hierarchy, brand consistency, composition, typography, color, affordance, information density) now appear in `critique-hardening.md` as a crosswalk: each dimension is a scan probe that feeds audit's existing five dimensions and P0-P3 severity. It is a lens set, not a second score, and brand-consistency checks require supplied brand references rather than inventing brand truth.

#### Perceived-quality lens, hardening, and evidence guard
A "Polish As Trust" lens ties perceived quality to observable consistency (spacing drift, grid misalignment, mismatched radii) and holds error/empty/loading states to the same craft bar as the primary path. Release-hardening gained component-completeness and accessibility-modality-coverage checks, plus a token-tier guard and a pseudo-localization build check. An evidence guard now bars conversion/behavior/business-impact claims unless supplied metrics, a baseline, or test evidence support them. Already-present content (RTL hardening, the Nielsen lens) was skipped.

## Interface and motion

#### Interface flow floor and copy
A compact UX quality floor (forms, search, navigation, feedback/error proximity, first-run/empty states) landed in `ux-quality-reference.md`; concrete copy formulas (error: what happened then why then what to do next; empty-state; CTA) and an earned-deviation restraint landed in the copy and principles references, with a build-facing media/illustration contract.

#### Async state machine and motion tokens
The genuinely net-new motion structure — an async state-machine card for branching UI (states, events, transitions, guards, impossible states, visible UI per state) — landed in `motion-pattern-cards.md`. Motion-token verification (named duration/easing tokens, a global reduced-motion override, explicit no-motion cases) and a gesture-accessibility rule landed in `motion-strategy.md`. Already-covered motion rules (the duration-band table, split/stagger, icon-swap, press-scale) were skipped.

## Foundations

A grid contract, density modes, and a containment restraint landed in `layout-responsive.md`; theme-specific media verification (logos, illustrations, screenshots per theme) and a cultural-color note landed in `palette-theming.md`; and a script-specific typography note landed in `typography-system.md`. The prior surface craft (concentric radius, image-edge outline, shadow-as-border, dark-mode ring, font smoothing, text-wrap, tabular numbers) and the existing data-visualization coverage were skipped.

## Ruled out

The corpus's lifecycle material — user research, UX strategy, validation programs, design-ops, governance, documentation systems, presentations, and negotiation — was recorded as out of scope, not adopted. No second scoring model, no global review format, and no per-mode logic in the hub. The cross-model sweep (Kimi-k2.7-code + DeepSeek-v4-pro) confirmed these rulings were sound and surfaced no missed major recommendation.
