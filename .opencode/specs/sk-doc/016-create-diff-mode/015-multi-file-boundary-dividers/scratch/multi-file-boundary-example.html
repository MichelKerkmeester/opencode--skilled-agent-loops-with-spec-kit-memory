1296+
1297+```bash
1298+ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519_work -C "git-account-work"
1299+```
1300+
1301+Add the public key to the intended provider account through its account
1302+settings. Do not upload or disclose the private key. Add the key to the local
1303+agent using the platform-appropriate keychain integration, for example:
1304+
1305+```bash
1306+# macOS
1307+ssh-add --apple-use-keychain ~/.ssh/id_ed25519_work
1308+
1309+# Linux or another agent-managed environment
1310+ssh-add ~/.ssh/id_ed25519_work
1311+```
1312+
1313+Add a host alias to `~/.ssh/config`:
1314+
1315+```sshconfig
1316+Host github-work
1317+ HostName github.com
1318+ User git
1319+ IdentityFile ~/.ssh/id_ed25519_work
1320+ IdentitiesOnly yes
1321+ AddKeysToAgent yes
1322+ UseKeychain yes
1323+```
1324+
1325+On clients without macOS `UseKeychain` support, omit that line and use the
1326+platform's SSH agent or keychain integration instead.
1327+
1328+For another provider, change both `Host` and `HostName` to provider-neutral
1329+values appropriate to that service. Configure only the selected repository:
1330+
1331+```bash
1332+git remote set-url origin git@github-work:<organization>/<repository>.git
1333+git config --local user.name "<commit-author-name>"
1334+git config --local user.email "<verified-commit-email>"
1335+```
1336+
1337+Authentication, authorship and remote selection are separate controls. Verify
1338+each without exposing personal values:
1339+
1340+```bash
1341+ssh -T git@github-work
1342+git remote get-url origin
1343+git config --local --get user.name
1344+git config --local --get user.email
1345+git ls-remote origin HEAD
1346+```
1347+
1348+The SSH handshake verifies the account selected by the alias. `git ls-remote`
1349+verifies repository reachability, not push permission. A rejected push may be
1350+caused by provider authorization, branch protection, a hook, non-fast-forward
1351+history or unrelated histories. Classify that error first; do not change keys
1352+or force-push as an authentication fix. For divergence, inspect safely with:
1353+
1354+```bash
1355+git fetch --prune origin
1356+git rev-list --left-right --count HEAD...origin/HEAD
1357+git merge-base HEAD origin/HEAD
1358+```
1359+
1360+If `git merge-base` returns no commit, stop and ask for a history-reconciliation
1361+decision. Never replace a remote branch merely because account selection is
1362+unclear.
