## Summary

`prompt_quality_card.md` carried a full copy of the sk-prompt framework table and CLEAR scoring rubric — content already maintained authoritatively in the sk-prompt skill. This bloat created drift risk and doubled maintenance surface. The card is now a lean Claude-specific supplement: model-defaults, permission-mode notes, and a pointer to the canonical source for everything else.

SKILL.md gains one prose block documenting the 3-tier precedence rule (sk-prompt canonical → cli-claude-code supplement → in-prompt override) and a cross-link to the sk-prompt hub so readers know where the full framework lives.
