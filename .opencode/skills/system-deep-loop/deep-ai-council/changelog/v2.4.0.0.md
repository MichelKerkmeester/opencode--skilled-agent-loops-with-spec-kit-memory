# v2.4.0.0 - Renamed to deep-ai-council

The mode packet folder `ai-council` is renamed to `deep-ai-council` so the folder name matches the SKILL.md `name`. That match is the invariant `package_skill.py --check` enforces, and it is the convention the rest of the deep-loop family already follows. The rename is structural only and preserves every caller-facing key.
