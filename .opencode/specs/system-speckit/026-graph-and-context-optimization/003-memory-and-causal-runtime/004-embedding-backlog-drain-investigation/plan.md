## 3. ARCHITECTURE

### Pattern
Evidence-first research packet with reducer-owned state and write-once iteration narratives.

### Key Components
- **Retry manager trace**: retention parking, queue selection, retry success write path.
- **Reindex trace**: `embedder_set`, vector-shard writes, active pointer flip, missing metadata success commit.
- **Daemon lifecycle trace**: launcher lease, IPC bridge, env loading, sidecar worker respawn.
- **Operator runbook**: reconcile active-vector-present stale rows before any missing-vector retry drain.

### Data Flow
1. Deep-research prompts dispatch bounded iteration work.
2. Iteration agents read code and prior packet evidence.
3. Findings land in `research/iterations/iteration-NNN.md`.
4. Canonical JSONL records append to `research/deep-research-state.jsonl`.
5. Structured deltas land in `research/deltas/iter-NNN.jsonl`.
6. The reducer synthesizes strategy, dashboard, registry, and final research output.
