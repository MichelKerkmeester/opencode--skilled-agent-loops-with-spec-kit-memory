## 2. PROBLEM & PURPOSE

### Problem Statement
The playbook's §2 global preconditions require a current MCP server build, an unset `SPECKIT_SKILL_ADVISOR_HOOK_DISABLED` flag, and evidence workspaces before any scenario can run. The CLI-delegated waves additionally require `cli-devin` and `cli-opencode` to be installed and authenticated. None of this had been confirmed for the current session.

### Purpose
Bring the environment to a known-good state so the 46 scenarios run against a current build, and confirm which executors are reachable.
<!-- /ANCHOR:problem -->

<!-- ANCHOR:scope -->
## 3. SCOPE

### In Scope
- Build `system-spec-kit` and `system-skill-advisor` MCP servers.
- Confirm `SPECKIT_SKILL_ADVISOR_HOOK_DISABLED` is unset.
- Create `/tmp/skill-advisor-playbook/` and `/tmp/devin-hook-playbook/` evidence workspaces.
- Probe `devin` and `opencode` availability + provider auth.

### Out of Scope
- Running any playbook scenario (those are phases 002-004).
- Installing missing CLI binaries.

### Files to Change

| File Path | Change Type | Description |
|-----------|-------------|-------------|
| `mcp_server/dist/**` | Build artifact | Regenerated by the build; not tracked |
