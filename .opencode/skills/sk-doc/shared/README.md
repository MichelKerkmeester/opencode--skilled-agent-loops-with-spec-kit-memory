# sk-doc/shared — create-quality-control backbone (scaffold)

Canonical home for the cross-cutting create-quality-control backbone consumed by every create-* packet: generic validator scripts, global standards/vocabulary references, and shared assets. Child packets symlink these inward; the sk-doc/{scripts,references,assets} root dirs keep facade symlinks pointing here so external references resolve unchanged.

NOT discoverable: no graph-metadata.json, no description.json here. Populated in the shared-backbone phase.
