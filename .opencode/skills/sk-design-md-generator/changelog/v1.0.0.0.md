# v1.0.0.0 - First Stable Release

**Released:** 2026-06-21

First stable release of the `sk-design-md-generator` skill. It captures a live website's **real, measured CSS** into a publication-quality 17-section `DESIGN.md` so AI agents build against ground truth instead of hallucinating colours, fonts, spacing, and shadows. The skill ships an embedded tool under `tool/` and runs a three-phase pipeline: **extract** (a Playwright crawler samples five viewports and emits `tokens.json`), **write** (compose the 17-section `DESIGN.md` copying every numeric value verbatim from `tokens.json`), and **validate** (`validate.ts` checks hex accuracy and section completeness). It is the extraction-and-format-fidelity engine of the `sk-design-*` family, sibling to `sk-design-interface` (which invents new distinctive direction) and the `mcp-figma` / `mcp-open-design` transports.

---

## What's New in 1.0.0.0

### Vendored working tool

- Embedded the extraction pipeline under `tool/`: 19 TypeScript pipeline modules + vitest tests, 6 knowledge docs in `tool/resources/`, and gold-standard `DESIGN.md` + `tokens.json` + `writing-notes.md` for stripe/vercel/linear/supabase in `tool/examples/`.
- Trimmed the regenerable example HTML reports (~2 MB) to keep the skill lean.

### Skill layer

- `SKILL.md` routes the three pipeline phases (extract / write / validate) plus a report and an example-study path, and encodes the cardinal verbatim-value rule, 6-digit lowercase hex, and the L1-L4 stability gates.
- `references/extraction_workflow.md` and `references/troubleshooting.md` add framework-specific operational guidance and failure-mode fixes; the deep design knowledge stays in `tool/resources/`.
- `INSTALL_GUIDE.md` documents the one-time `cd tool && npm install && npx playwright install chromium` setup and a first extraction.

### Advisor registration

- `graph-metadata.json` registers the skill (family `sk-code`, category `design`) with reciprocal sibling edges into the `sk-design-*` family (`sk-design-interface`, `mcp-figma`, `mcp-open-design`) and an `enhances` edge to `sk-code` (the `DESIGN.md` is the implementation contract sk-code builds against).

---

## Cardinal Rule

Every hex, pixel, font-weight, shadow, and radius in a `DESIGN.md` must be copied verbatim from `tokens.json`. A value that is estimated, rounded, or invented breaks the document's entire reason to exist - a hallucination-proof reference. Validation runs before any completion claim.
