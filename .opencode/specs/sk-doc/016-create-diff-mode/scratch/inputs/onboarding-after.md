# Onboarding Guide

Welcome to the platform. This short guide walks new users through account setup and first login.

## Prerequisites

- A supported browser (Chrome, Firefox, or Safari)
- An account invitation from your workspace admin

## Installation

Run the installer and follow the prompts. The process usually takes about five minutes.

## First Login

After installation, sign in with the credentials from your invitation email.

## Support

Email support@example.com or open a ticket in the help center if you run into trouble.
