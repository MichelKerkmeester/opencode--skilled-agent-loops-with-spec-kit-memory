## Program Theme

The two releases are one story told in two registers. system-spec-kit v3.7.0.0 is the wide pass: the verdict honesty work, the dark-flag reckoning that measured eight families and graduated four, the bitemporal edge history that a deep review caught corrupting live reads and a repair closed, then the late-cycle surfacing fix where a real query that should have matched many spec folders was returning one until the trigger lane was promoted and budget overflow was degraded into compact rows. system-skill-advisor v0.10.0 is the narrow pass on the same principle: a routing field that reads untrusted text now gets sanitized before it can steer a recommendation, with the durable metrics writer and a couple of async-safety defects closed alongside.

The connective tissue is the shipping rule. A structural protection that only ever makes the system safer ships on for everyone, anything that changes a result ships off behind a named flag until a benchmark earns it on, and a known remaining gap gets written down rather than left unstated. Both releases honor it, and both were verified by reading the real path rather than trusting the test that passed.

For the detail, read the two skill changelogs. This aggregate is an index, not a re-narration.
