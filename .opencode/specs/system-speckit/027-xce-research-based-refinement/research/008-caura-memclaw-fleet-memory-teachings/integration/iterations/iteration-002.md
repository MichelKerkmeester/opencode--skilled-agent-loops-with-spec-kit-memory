## New skill needed?

No. This is hardening of an existing local memory system, not a new domain. `system-spec-kit` already owns Spec Kit Memory and MCP runtime behavior, `sk-doc` owns documentation quality, `sk-code` owns OPENCODE implementation/verification, and `system-skill-advisor` owns routing only.
