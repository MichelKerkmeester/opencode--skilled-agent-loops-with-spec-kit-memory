## What Was Built

Status COMPLETE. Five recommendations landed in the real scoring sources, four as default-OFF behavioral flags and one as documentation. With every flag OFF the band, the envelope and the citation policy reproduce the shipped output, so the default runtime is unchanged until a flag graduates against the off-corpus fixtures.

The real source under `mcp_server` was edited, not the compiled `dist` declaration the scaffold pointed at. The typed `.ts` is the source of truth and the `dist` `.js`/`.d.ts` were regenerated by `npm run build`, so the typed and compiled surfaces match.

### Grounding signal in the envelope (rec 7)

A `grounding` field (`grounded` / `low_grounding`) is surfaced on the result envelope, built from the lexical overlap already on the rows by the new `assessGrounding` in `confidence-scoring.ts`. The formatter attaches it at the verdict-field population behind `SPECKIT_GROUNDING_SIGNAL_V1`. It reuses the `fts_score` / `bm25` signals already present and adds no recompute. With the flag OFF no grounding field is added, byte-for-byte the shipped shape.

### Noise-floor subtraction before banding (rec 8)

`assessRequestQuality` subtracts a measured corpus noise-floor from the absolute relevance before the band read, floored at zero, behind `SPECKIT_NOISE_FLOOR_SUBTRACTION_V1`. The floor lives in the new `noise-floor.ts` keyed by the embedder it was measured against (`nomic-embed-text-v1.5`), and an embedder with no measured floor resolves null and fails closed to the raw band. The margin is computed off the raw scores, where a constant floor cancels, so it stays identical. With the flag ON the off-corpus high-cosine sample drops from good to weak; with it OFF the band reads raw relevance unchanged.

### cite_with_caveat tier (rec 10)

`deriveCitationPolicy` in the formatter adds a `cite_with_caveat` tier between `cite_results` and `do_not_cite_results`, behind `SPECKIT_CITE_WITH_CAVEAT_V1`. A weak verdict whose top hit is still lexically grounded and clears the weak relevance floor is hedged; a clear good stays `cite_results` and a clear miss stays `do_not_cite_results`. A fully ungrounded hit is never promoted. With the flag OFF the policy is the shipped two-state output.

### Evidence-gap bridge into the verdict (rec 11)

`assessRequestQuality` reads an `evidenceGapDetected` option the formatter threads in from the same Stage 4 `evidenceGap` signal the recovery path reads, behind `SPECKIT_EVIDENCE_GAP_VERDICT_V1`. A true gap caps a good verdict at weak. The gap is read from the passed signal, never recomputed. With the flag OFF the verdict ignores the gap exactly as today.

### Calibration re-fit documented as a non-fix (rec 12)

The band label is taken off the PRE-calibration value: `confidence-scoring.ts:531` bands on `rebalancedValue` while the isotonic model is applied separately to the displayed `confidence.value` at `:519`, and the request-quality verdict bands on `topScore` derived from the pre-calibration `rawTopScore` at `:585`. A curve re-fit therefore cannot move good versus weak versus gap, so the phase ships the documented non-fix statement in `ENV_REFERENCE.md` and an executable proof in the test, and edits no calibration curve and no `maybeCalibrate`.

## How It Was Delivered

The four flag readers were added with the opt-in helper already used by the sibling lexical-grounding and envelope-fidelity flags. The noise-floor module was added as the measured-floor source of truth. The band subtraction and the evidence-gap cap landed in `assessRequestQuality`, restructured to compute a single label and return once so the cap composes cleanly while every flag-OFF path stays byte-identical. The grounding assessment and the caveat tier landed in the formatter, reusing the lexical signals on the rows. The typed sources were compiled to `dist` with `npm run build`. A 16-case vitest pairs every flag-ON case with a flag-OFF case asserting the shipped output, and proves the noise-floor records its embedder, fails closed for an unmeasured embedder, and floors at zero. The rec-12 non-fix is proven executably by asserting the verdict label is identical with confidence calibration ON and OFF.

<!-- ANCHOR:decisions -->
## Key Decisions

| Decision | Why |
|----------|-----|
| Edit the real `.ts` sources and rebuild `dist`, not the `dist` declaration the scaffold listed | The typed source is the source of truth and the compiled surface is generated, so editing `dist` directly would drift from the next build |
| Ship every behavioral change behind a default-OFF flag | The existing eval and verdict fixtures carry the shipped shapes the new contract changes, so the OFF arm must reproduce the shipped output until a flag graduates against the off-corpus fixtures |
| Compute the margin off raw scores under noise subtraction | A constant floor cancels in a first-minus-second margin, so reading raw keeps the margin byte-identical and avoids a spurious change |
| Key the noise-floor by embedder and fail closed | The background cosine is embedder-specific, so a floor measured on one embedder is meaningless on another and an unmeasured embedder must not subtract an unknown value |
| Add a middle cite_with_caveat tier rather than widen the weak drop | A borderline grounded near-miss and a clear miss are not the same signal, so hedging is more faithful than dropping both |
| Read `stage4.evidenceGapDetected` rather than recompute the gap | The gap is already computed in Stage 4, so bridging the existing signal avoids a second source of truth |
| Document the calibration re-fit as a non-fix instead of attempting it | The verdict band is provably independent of the curve, so a re-fit cannot move the verdict and would spend effort on a non-fix |

## Known Limitations

1. **Flags stay OFF until their off-corpus arm graduates.** The behavioral changes ship dark; a flag graduates only after the off-corpus hard-negative fixtures and the wired false-confirm metric confirm its arm is green while the aligned good queries and the correctly-weak case hold.
2. **Embedder-portability.** The corpus noise-floor is measured against a single embedder (`nomic-embed-text-v1.5`). The rec-8 flag fails closed to the raw band for any embedder with no measured floor, and a second embedder needs its own measured entry before it subtracts.
3. **Active-embedder resolution.** The noise-floor resolver defaults to the local-first embedder and accepts an explicit embedder argument; wiring the live active-embedder identity into the verdict call is deferred to graduation, when the floor is read for the embedder actually in use.
4. **Root-cause dependency.** This phase surfaces, corrects the floor, hedges, and bridges around the convergent lexical-grounding floor and single-hit corroboration, which ship in their own phases, so the hardening is most effective once those root-cause fixes also land.
