**Released**: 2026-05-17

## Summary

This addendum records two follow-on findings that together close the exploration of whether the v1.0.5.0 RCAF-default decision should change:

1. **File-extraction unlock** — the 113/005 packet built a markdown-to-disk extraction layer (the caveat called out at the end of v1.0.5.0 §Upgrade), re-ran the eval-loop with extraction enabled, and confirmed RCAF still wins on aggregate scoring. The 5-dim rubric's D1 signal is now unlocked but does not flip the ranking.

2. **CONTEXT-framework mutation, single-sample win, NOT REPRODUCED** — an 8-iter mutation-depth re-run produced a CONTEXT-framework variant (`v-mut-d68b487314246cd3`) scoring 0.5833 vs the v2 RCAF baseline of 0.5664 (+1.7%). Below the +2% confidence threshold. A 3-run confirmation against the same 7 fixtures landed at mean 0.5252, **below the RCAF baseline by 4.1%**. The CONTEXT win was run-to-run noise, not a genuine framework-on-surface finding.

The composite verdict: RCAF + medium pre-plan + standard bundle-gate + standard anti-hallucination remains the empirically-best SWE-1.6 default. v1.0.6.0 not shipped.

## Empirical evidence

### File-extraction layer (113/005 packet)

The v1.0.5.0 §Upgrade caveat noted that SWE-1.6 `--print` mode returns markdown rather than files on disk, capping D1 Acceptance to 0.0 on 4 of 7 fixtures via the D2 hard-gate. The 113/005 packet built `scripts/extract-files-from-markdown.cjs` — a 4-pattern path-inference extractor (md-header-backticked, md-header-bare, backticked-line, bold-line + first-line-comment fallback) — and wired it into score-variant.cjs behind `EVAL_LOOP_EXTRACT=true`.

Re-running the 5 council-seeded variants with extraction enabled (v2 run): RCAF still wins at 0.5664. Extraction unlocked D1 partially but did not change the aggregate ranking. Per-fixture analysis revealed that v-001 (STAR) and v-005 (BUILD-strict) each dominate distinct fixture clusters — RCAF wins the average case but isn't strictly dominant per-fixture. The aggregate-vs-per-fixture distinction is recorded in `specs/skilled-agent-orchestration/113-cli-devin-prompt-quality-arc/005-extraction-rerun/per-fixture-analysis.md`.

### CONTEXT mutation re-run (v3)

To probe whether mutation-depth could surface a better variant beyond the v2 RCAF plateau, the eval-loop ran 3 additional hill-climb iterations beyond the v2 cap (`--max-iters=8`). The mutator produced `v-mut-d68b487314246cd3` with framework=CONTEXT, preplanning_density=medium, thinking_threshold=5, bundle_gate_strictness=standard, anti_hallucination_strength=standard — scoring 0.5833 (+1.7% over RCAF). Recorded in `specs/skilled-agent-orchestration/113-cli-devin-prompt-quality-arc/005-extraction-rerun/synthesis-v3.md`.

The +1.7% delta sat below the synthesize-v3 ≥0.02 confidence threshold. Run-to-run variance on the same fixtures across v1/v2/v3 had been measured at 0.02–0.15 (v-005 ranged 0.4139–0.5610 across runs), so the single-sample CONTEXT win was within typical noise.

### Confirmation run (3 × 7 fixtures, REAL SWE-1.6 + claude-sonnet grader)

Dispatched `scripts/confirm-variant.cjs` against `v-mut-d68b487314246cd3` for 3 independent runs of the same 7-fixture set. Results:

| Run | Aggregate score | Δ vs v3 single-sample (0.5833) | Δ vs v2 RCAF baseline (0.5664) |
|-----|-----------------|---------------------------------|--------------------------------|
| 1 | 0.4985 | −0.0848 | −0.0679 |
| 2 | 0.5758 | −0.0075 | +0.0094 |
| 3 | 0.5013 | −0.0820 | −0.0651 |

**Mean: 0.5252  |  Std: 0.0358  |  Range: 0.0774  |  Mean lift over v2 RCAF: −0.0412**

Per-fixture variance across 3 runs was substantial — fix-001 swung 0.530 → 0.817 → 0.530 (range 0.287); fix-004 ranged 0.662 → 0.838; fix-006 swung 0.330 → 0.530. The high intra-fixture variance confirms that single-iteration mutation results on this fixture set are unreliable. Recorded in `specs/skilled-agent-orchestration/113-cli-devin-prompt-quality-arc/005-extraction-rerun/confirmation-report.md`.

The confirmation runner uses verdict thresholds: REPRODUCED requires all runs ≥ v2 RCAF baseline AND mean within 0.005 of v3 target; ROUGHLY_REPRODUCED requires mean lift ≥0.02 over v2 RCAF. The CONTEXT mutation failed both gates.

**Verdict: NOT_REPRODUCED.** v1.0.6.0 not shipped.

## Operational guidance (unchanged)

RCAF + medium pre-plan + standard bundle-gate + standard anti-hallucination remains the SWE-1.6 default for cli-devin dispatches. The four v1.0.5.0 empirical findings stand:

1. RCAF wins; STAR for narrative-heavy, BUILD for multi-file refactors.
2. Medium pre-plan density; do not go dense.
3. Standard bundle-gate language; do not tighten beyond standard.
4. Framework choice dominates anti-hallucination wording.

For future mutation-driven exploration on this surface: a single +N% improvement at N<2% should not be treated as a confirmed win. Run 3 confirmation iterations against the same fixture set before promoting a mutated variant to default. The fixture-set's intrinsic run-to-run noise floor is ~0.04–0.08 on aggregate score.
