# Layout And Adaptation Capability

## Description

Builds spacing, hierarchy, grid, and responsive rules that make a static system usable across contexts.

## Current Reality

The skill uses spacing as the first hierarchy tool, chooses flex or grid by dimensional need, avoids card-grid monotony, and adapts to viewport, input method, orientation, safe areas, and content pressure.
