# Color Systems Capability

## Description

Builds and repairs static color systems with OKLCH, semantic roles, contrast gates, gamut checks, and theme mapping.

## Current Reality

The skill applies OKLCH for perceptual control, assigns roles before values, fixes contrast by changing lightness first, treats dark mode as a separate surface system, and rejects color that has no semantic, hierarchy, wayfinding, or brand role.
