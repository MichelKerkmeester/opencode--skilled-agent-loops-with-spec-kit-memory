## 3. ARCHITECTURE

### Pattern
**Mechanical refactor across a fixed live-surface allow-list.** No new code; coordinate of edits + advisor reindex.

### Key Components
- **Skill body** (`.opencode/skills/sk-prompt-small-model/`): The thing being renamed. Internal references (SKILL.md frontmatter, README.md title, description.json, graph-metadata.json `skill_id`, references/pattern-index.md self-references, changelog/v0.3.0.0.md new-file).
- **Sibling skill graph metadata** (cli-devin + cli-opencode `graph-metadata.json`): live `edges.enhances[].target` + `manual.related_to[]` references that route the advisor to the sentinel skill.
- **Manual testing playbooks** (cli-devin + cli-opencode `manual_testing_playbook/`): operator-facing how-to docs that reference the skill by name in filename + body.
- **Root behavioral docs** (AGENTS.md + CLAUDE.md + README.md): the "Small-model dispatch rule" + the skill catalog entry.
- **Auto-memory current-state entries** (~/.claude/projects/.../memory/MEMORY.md + dispatch-matrix memory entry + skill-graph-rebuild feedback entry): durable cross-session pointers that must reflect the current canonical name.
- **Compiled skill-graph index** (`.opencode/skills/system-skill-advisor/mcp_server/scripts/skill-graph.json`): regenerated artifact, never hand-edited.
- **Phase-parent metadata** (`114/spec.md` PHASE DOCUMENTATION MAP + `114/graph-metadata.json` derived/children_ids): forward-pointing note + child registration.

### Data Flow
```
git mv sk-small-model → sk-prompt-small-model
  ↓
Edit skill body files (frontmatter, README, description.json, graph-metadata.json, pattern-index)
  ↓
Edit sibling skill graph-metadata.json (cli-devin + cli-opencode enhances/related_to edges)
  ↓
Edit sibling manual playbooks (rename files + edit body refs)
  ↓
Edit root docs (AGENTS.md, CLAUDE.md, README.md)
  ↓
Edit auto-memory (MEMORY.md index + matrix entry + rebuild-feedback entry)
  ↓
Run skill_graph_compiler.py --export-json --pretty (rebuilds skill-graph.json + skill-graph.sqlite via launcher)
  ↓
advisor_validate / advisor_recommend smoke test
  ↓
Update parent 114/spec.md PHASE DOCUMENTATION MAP + 114/graph-metadata.json via generate-context.js (Step 13)
```
