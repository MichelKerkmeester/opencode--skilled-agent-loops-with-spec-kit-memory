# v1.0.0.0 - Initial Release

**Released:** 2026-06-25

Initial release of `audit`, the cross-cutting design QA and critique child in the `sk-design` family.

## What's Included

- `SKILL.md` with independent routing boundaries for design audit, accessibility, performance, responsive, theming, critique, anti-slop, and production hardening prompts.
- P0-P3 severity model and five-dimension `/20` scoring contract aligned in spirit with `sk-code-review`.
- Focused references for audit contract, accessibility/performance, critique/hardening, anti-patterns/production details, and corpus traceability.
- Lean `feature-catalog/` and `manual-testing-playbook/` packages.
- `graph-metadata.json` with family and sibling edges.

Distills `external/audit.md`, `external/critique.md`, `external/polish.md`, `external/harden.md`, `external/optimize.md`, `external/fixing-accessibility.md`, `external/fixing-motion-performance.md`, `external/clarify.md`, and `external/pseudo-elements.md`.
