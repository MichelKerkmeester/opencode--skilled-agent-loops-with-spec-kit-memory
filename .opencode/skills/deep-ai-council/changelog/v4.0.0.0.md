## Deep Mode Behavior

Deep mode uses the packet 129 state hierarchy:

```text
session
  -> topic
     -> round
```

Each topic runs bounded council rounds and stops on adjudicator-verdict stability, max rounds, or failure conditions. The session reducer writes `deep-ai-council-findings-registry.json`, and later topics receive compact cross-topic priors by registry fingerprint.

Default cost guards remain conservative:

- `max_rounds_per_topic = 3`
- `max_topics_per_session = 5`
- `saturation_threshold = 0.20`
- `seats_per_round = 3`
