## Summary

Matured `sk-prompt-small-model` from a thin routing sentinel into the canonical per-model prompt-craft **content hub** (Architecture A). The skill now owns the substantive prompt-craft knowledge for every active small model; executor mechanics (binary flags, invocation wrappers) remain in `cli-devin` and `cli-opencode`. The SKILL.md version field, which had drifted to `0.2.0` while the changelog reached v0.4.0.0, is realigned to `0.5.0.0`.
