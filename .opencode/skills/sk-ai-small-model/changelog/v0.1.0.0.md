## [**0.1.0.0**] — 2026-05-18

This first release introduces `sk-small-model` as the sentinel routing anchor for small-model optimization patterns across `cli-devin` and `cli-opencode` dispatch surfaces. The skill keeps real patterns distributed in the executor skills (HYBRID-with-Anchor architecture) while giving the skill advisor one discoverable entry point for any prompt that names a small model or a small-model pattern.

> Spec folder: `.opencode/specs/skilled-agent-orchestration/114-small-ai-model-optimization/002-foundation-routing/` (Level 2). Research synthesis: `001-research-smallcode/research/research.md` (1204 lines, HYBRID-with-Anchor verdict).

---

## What Changed

#### Sentinel skill scaffold

The sentinel ships as a thin routing anchor: `SKILL.md` (≤ 200 LOC), `references/pattern-index.md` (≤ 100 LOC), `description.json` + `graph-metadata.json` with `enhances` edges (weight 0.5) to both `cli-devin` and `cli-opencode`, and a sk-doc-aligned `README.md`. No runtime logic, no scripts, no agent-config recipes — every pattern body lives in the owning executor skill. Operators reach the canonical pattern files through one Read of `references/pattern-index.md`.

---

#### Dispatch matrix coverage

Active small-model rotation: SWE-1.6, DeepSeek-v4-pro, Kimi-k2.6, Qwen3.6. Optional unverified separate-pool stubs: Claude Haiku (Anthropic), Gemini Flash (Google). Frontier models (Opus, Sonnet, gpt-5.5, GLM-5.1) explicitly out of scope.

---

#### AGENTS.md + CLAUDE.md dispatch rule

Adds the `Small-model dispatch rule` to the canonical `AGENTS.md` (auto-synced to `CLAUDE.md` via symlink). Operators MUST consult `sk-small-model` before dispatching to any small model, in addition to the per-CLI `cli-X/SKILL.md` read required by the existing CLI dispatch rule. Routing is automatic via `enhances` edges; the prose rule is a backstop for cases where the advisor doesn't auto-surface.

---

#### Advisor wiring + reverse edges

`graph-metadata.json` declares `enhances` edges from `sk-small-model` to both `cli-devin` and `cli-opencode` at weight 0.5; reverse `enhances` edges are added to both `cli-devin/graph-metadata.json` and `cli-opencode/graph-metadata.json` so the graph_causal lane fires from either direction. After the skill-graph compiler runs, the advisor surfaces `sk-small-model` at confidence ≥ 0.85 on all small-model dispatch prompts tested.

---

#### Root README + 21-skill count

`README.md` (repo root) lifts skill count from 20 → 21 in 5 places and adds a `sk-small-model` entry under §3 OTHER with active scope, advisor wiring, trigger phrases, and the pattern-index map.
