# Motion Strategy Capability

## Description

Converts motion requests into purpose, budget, timing, easing, staging, and material decisions.

## Current Reality

The skill requires each animation to justify itself as feedback, orientation, focus, continuity, perceived performance, or delight. It uses fast timing for interaction, slower timing only for earned choreography, and avoids scattered reveals.
